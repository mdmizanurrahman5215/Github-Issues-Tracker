## 1.What is the difference between var, let, and const?
### var : 
- It is  Functional scope.
- we can reassign and redeclare it
### let :
- it is  block scope.
- we can reassign but can't redeclare.
### const :
- It also block scope .
- we can't reassign and redeclare it
